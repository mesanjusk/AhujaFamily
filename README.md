# AhujaFamily Monorepo

This monorepo contains the frontend and backend for the AhujaFamily project.

## Structure

- `frontend/` — React + Vite frontend (originally mesanjusk/frontend)
- `backend/` — Node.js backend (originally mesanjusk/backend)

## Getting Started

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
node index.js
```
